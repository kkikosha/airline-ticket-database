# Airline Ticket Database System
![SQL](https://img.shields.io/badge/SQL-Database-blue)

## Overview

This project designs and implements a relational database system for managing airline ticket sales and flight operations.

The database stores information about flights, passengers, aircraft, tickets, employees, and airline operations. The system allows efficient storage, querying, and analysis of airline ticket sales data.

## Project Goal

The goal of the project was to design a structured relational database that supports airline ticket management and provides tools for data analysis and reporting.

## Technologies

- SQL

## Database Design

The database includes multiple interconnected entities such as:

- airports  
- airlines  
- aircraft  
- flights  
- passengers  
- tickets  
- operations  
- employees  

The database schema was normalized to **Third Normal Form (3NF)** to ensure data consistency and eliminate redundancy.

## SQL Features

The project includes several database development components:

- complex SQL queries  
- JOIN operations  
- GROUP BY aggregations  
- subqueries  
- indexes for performance optimization  
- stored procedures  
- triggers  
- views  

These components enable efficient data retrieval and automated database operations.

## Reports and Visualization

The project includes analytical reports built using:

- **Excel** for tabular reports and analysis  
- **Power BI** for data visualization and dashboards  

These tools allow users to analyze ticket sales, flight operations, and other airline performance indicators.
